#### 安装
```bash

# or
yarn add @cooj/utils
```

#### 方法
- 获取时间,并格式化
```ts
import { formatTime } from '@cooj/utils'

formatTime(new Date(), 'YYYY-mm-dd HH:MM:SS') // 2021-12-29 09:46:08
```
